# neetcoder

to run use

`npm install`
then `npm run dev`
to start the bot.

Use `docker compose up` to spin up the database.

You will need to create your own bot the set the `CLIENT_ID`, `CLIENT_SECRET` and `GUILD_ID`.

https://discordjs.guide/legacy/app-creation/project-setup
